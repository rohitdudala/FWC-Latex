$F(X, Y, Z) = (X' + Y + Z)\cdot(X + Y' + Z') \cdot (X' + Y + Z') \cdot (X'Y'Z' + X' Y Z' + XYZ')$.

Which one of the following is true?
\hfill(IN2021,31)

\begin{enumerate}
    \item $F(X, Y, Z) = (X + Y + Z') \cdot (X' + Y' + Z')$
    \item $F(X, Y, Z) = (X'+ Y) \cdot (X + Y' + Z')$
    \item $F(X, Y, Z) = X'Z' + YZ'$
    \item $F(X, Y, Z) = X'Y'Z + XYZ$
\end{enumerate}
\end{enumerate}
